# mas-lab-content

Shared content engine for lab tutorials and documentation: Markdown with widget directives, export to slides/HTML, and reusable content blocks.

**Allowed dependencies:** stdlib, `pyyaml`, `mas-lab-core` (paths/schemas only). Must not import `mas.ctl`, `mas.runtime`, or benchmark scheduler modules.
